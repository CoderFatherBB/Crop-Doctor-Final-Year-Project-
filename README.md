## About

Crop Doctor is a final-year project for crop disease detection and farmer assistance. The repository contains the Flask app, templates, static assets, trained Keras models, upload flow, admin model-training screen, and chatbot interface.

## Repository Map

- `app.py`: Flask application entry point.
- `templates/`: login, upload, dashboard, chatbot, train-model, admin dashboard, and error pages.
- `static/`: CSS, images, and upload assets.
- `models/`: trained model files and chatbot resources.
- `App_Images/`: screenshots used in this README.
- `requirements.txt`: Python dependencies.
