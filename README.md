## 📖 Overview
Crop Doctor is an AI-powered agricultural assistant capable of diagnosing plant diseases from images with extreme precision. Validated across 4 distinct datasets including Sugarcane Leaf and Coconut Tree diseases, it achieves a peak F1-score of 0.94.

## ✨ Core Features
- **Multi-Crop Support**: Capable of detecting numerous specific pathogens.
- **Edge Deployment Ready**: Optimized CNN inference allowing deployment on low-power devices.
- **Data Augmentation Engine**: Highly robust training pipeline dealing with varying lighting and noisy agricultural captures.

## 🛠️ Architecture
Built heavily on deep Convolutional Neural Networks (CNNs) using TensorFlow/Keras, featuring transfer learning from architectures like ResNet50 and MobileNetV2 optimized for mobile inference.
